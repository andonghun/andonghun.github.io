    # 개발환경 for FE Developer

    # Editor

    주력 에디터는 자동완성 정도만 제공해주는 VS Code가 괜찮다.

    vi, vim 정도는 익혀놓자. 장비가 없는 상황에서 터미널을 통해 서버에 들어가서 급하게 수정해야 할 때 사용하는 데에 유용하다.

    # Source Code Management

    회사에서 제공해주는 데로 사용해야 한다.

    하지만 git이 유용하다.

    특히 git에서 commit message를 작성할 때 습관적으로 훌륭하게 작성하자.
